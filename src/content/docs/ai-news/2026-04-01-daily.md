## 论文精选

### TAPS: 投机采样中的任务感知提议分布
**👍 116** · [arXiv](https://arxiv.org/abs/2603.27027)

本文探讨了投机解码（Speculative Decoding）中，草稿模型（Draft Model）的训练数据分布对最终生成质量和加速比的影响。工程启发：为大模型推理加速提供了新思路，在部署投机解码时，针对特定业务场景微调草稿模型能显著提升 Token 接受率和系统吞吐量。

### Gen-Searcher：强化图像生成的智能体搜索能力
**👍 43** · [arXiv](https://arxiv.org/abs/2603.28767)

提出了一种训练搜索增强图像生成模型的方法，以解决模型内部知识冻结、难以应对需要最新信息场景的问题。工程启发：将 RAG（检索增强生成）和 Agent 工具调用机制成功引入多模态生成领域，为构建需要实时外部知识的图像生成系统提供了架构参考。

### 生成式多智能体系统中的涌现性社会智能风险
**👍 41** · [arXiv](https://arxiv.org/abs/2603.27771)

研究了多个大语言模型 Agent 在共同规划、谈判和分配资源时，产生的无法简单归因于单个智能体的集体故障模式。工程启发：在设计多智能体协作架构（如多 Agent 软件开发或自动化运维）时，必须引入系统级的安全护栏和死锁检测机制，而不能仅依赖单体 Agent 的对齐。

### Trace2Skill：将轨迹局部经验蒸馏为可迁移的智能体技能
**👍 41** · [arXiv](https://arxiv.org/abs/2603.25158)

通过自动提取 Agent 运行轨迹中的经验，生成可泛化的领域特定技能，解决了手动编写技能带来的扩展性瓶颈。工程启发：为大模型 Agent 技能库的构建提供了一条自动化路径，非常适合用于开发能够根据历史任务自我进化和沉淀工具集的 Agent 系统。
