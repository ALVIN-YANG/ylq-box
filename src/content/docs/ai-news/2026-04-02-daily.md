## 论文精选

### FIPO：通过未来 KL 影响策略优化激发深度推理能力
**👍 290** · [arXiv](https://arxiv.org/abs/2603.19835)

本文提出了一种新型强化学习算法 FIPO，旨在解决大语言模型的推理瓶颈问题，克服了传统 GRPO 算法中基于结果的奖励（ORM）在细粒度信用分配上的缺陷。对工程实践的启发在于，它为训练具有长逻辑链推理能力的 Agent 提供了更高效的 RLHF 策略参考，有助于提升模型在复杂任务中的规划成功率。

### GEMS：具备记忆与技能的 Agent 原生多模态生成框架
**👍 62** · [arXiv](https://arxiv.org/abs/2603.28088)

受 Claude Code 等先进 Agent 框架启发，研究团队提出了一种结合记忆机制和外部工具技能的多模态生成框架。这为工程团队构建不仅能“看”和“说”，还能通过调用工具和检索记忆来完成复杂长程任务的多模态 Agent 提供了优秀的架构范例。

### LongCat-Next：将多模态词汇化为离散 Token
**👍 117** · [arXiv](https://arxiv.org/abs/2603.27538)

该研究打破了现有系统将非文本模态作为外部附件的局限，通过离散原生自回归模型实现了真正的多模态统一。对于多模态 Agent 基础设施建设而言，这种原生融合架构能显著降低跨模态对齐的工程复杂度与推理延迟，是未来端到端多模态 Agent 的重要演进方向。
