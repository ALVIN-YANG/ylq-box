## 开源热门
> 数据来源：[Trendshift](https://trendshift.io/) · [GitHub Trending](https://github.com/trending)

### [block/goose](https://github.com/block/goose)
**⭐ 35,868 · Rust**

一款开源且可扩展的 AI Agent，不仅提供代码建议，还能通过任何 LLM 执行安装、运行、编辑和测试任务。该项目采用 Rust 编写，为开发者提供了一个底层且高度可定制的 Agent 执行环境，非常适合需要将大模型能力深度集成到本地开发工作流中的工程场景。

### [tirth8205/code-review-graph](https://github.com/tirth8205/code-review-graph)
**⭐ 4,891 · Python**

为 Claude Code 构建本地代码知识图谱的工具，通过持久化代码库映射让模型只读取关键上下文。从工程角度看，这是一种极具参考价值的代码级 RAG 优化策略，能显著降低代码审查（6.8倍）和日常编码（49倍）的 Token 消耗，适合解决超大代码库的上下文窗口瓶颈。

### [anthropics/claude-code](https://github.com/anthropics/claude-code)
**⭐ 109,016 · Shell**

运行在终端的 Agentic 编码工具，能通过自然语言命令理解代码库、执行日常任务并处理 Git 工作流。这是官方下场的重磅级终端 Agent，其架构设计和对本地环境的无缝集成方式，为构建下一代 CLI 开发者工具提供了标准范式。

### [msitarzewski/agency-agents](https://github.com/msitarzewski/agency-agents)
**⭐ 71,854 · Shell**

提供开箱即用的完整 AI 智能体团队，每个 Agent 都具备特定的人格、流程和可验证的交付能力。该项目展示了多智能体协同（Multi-Agent）在实际业务中的落地模式，适合需要快速搭建具有特定领域专家角色的复杂 AI 系统的开发者。

## 论文精选

### SKILL0：用于技能内化的上下文 Agentic 强化学习
**👍 83** · [arXiv](https://arxiv.org/abs/2604.02268)

本文解决大模型 Agent 在推理时动态加载外部技能导致的检索噪声和 Token 开销问题。通过引入上下文强化学习机制，让模型真正“内化”这些程序性知识，为降低 Agent 运行成本和提升工具调用稳定性提供了新的训练思路。

### CORAL：面向开放式发现的自主多智能体进化框架
**👍 41** · [arXiv](https://arxiv.org/abs/2604.01658)

现有的基于 LLM 的进化方法严重依赖固定启发式规则，限制了 Agent 的自主性。CORAL 提出了首个用于开放式问题的自主多智能体进化框架，对设计具备长期探索能力和知识积累的 Multi-Agent 架构具有重要参考价值。

### DataFlex：以数据为中心的大语言模型动态训练统一框架
**👍 172** · [arXiv](https://arxiv.org/abs/2603.26164)

该论文提出了一个统一的框架，用于优化 LLM 训练过程中的数据选择、混合和重加权，解决了现有代码库接口不一致的问题。对于从事大模型预训练或微调的基础设施工程师而言，这提供了一套标准化的高效数据处理流水线设计方案。
