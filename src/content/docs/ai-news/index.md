面向 AI Agent / 基础设施工程师的技术日报，通过 GitHub Actions 定时采集、LLM 自动总结生成。

**数据源**：

- **[Trendshift](https://trendshift.io/)** — 每日社区热门 AI 开源项目（基于参与度评分算法，优于原始 GitHub Trending）
- **[Hugging Face Daily Papers](https://huggingface.co/papers)** — 社区投票筛选的高质量 AI 论文
- **Release Watchlist** — 可配置的核心项目版本追踪（LangChain、vLLM、Ollama、OpenClaw 等）
- **Devtools Watchlist** — AI 开发者工具版本追踪（Cursor、Aider、Continue、Goose 等）
- **RSS Feeds** — Hacker News AI、TechCrunch AI、Hugging Face Blog、Simon Willison Blog、Cursor Blog

**栏目**：

- **开源热门** — GitHub Trending 中值得关注的 AI 项目
- **论文精选** — 当日高票 AI 论文，重点关注 Agent/RAG/推理方向
- **版本更新** — AI 框架关注列表中的新 Release（无更新时省略）
- **开发者工具** — AI 编码工具的版本更新（无更新时省略）
- **行业动态** — 筛选后的行业新闻与技术博文

**更新频率**：
- 每日 8:00（北京时间）自动发布技术日报
- 每周日自动生成深度技术周报
